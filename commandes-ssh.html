Hostinger — SSH

Commandes SSH
de Base

Les commandes SSH essentielles que tout administrateur doit connaître pour gérer un serveur distant en toute sécurité.

Introduction

Qu'est-ce que SSH ?

SSH (Secure Shell) est un protocole réseau cryptographique permettant d'accéder à un serveur distant de manière sécurisée. Il chiffre toutes les données échangées entre le client et le serveur, ce qui en fait le standard pour l'administration de serveurs.

Avantages de SSH

  • Connexion chiffrée de bout en bout
  • Authentification par clé cryptographique
  • Transfert de fichiers sécurisé
  • Tunneling et redirection de ports

Prérequis

  • Un hébergement avec accès SSH (Hostinger)
  • Un client SSH (Terminal ou PuTTY)
  • Les identifiants de connexion
  • L'adresse IP du serveur
Connexion

Se Connecter en SSH

La première étape est de se connecter à votre serveur via SSH depuis votre terminal.

ssh — Connexion à un serveur distant

Établit une connexion sécurisée avec votre serveur. Remplacez les valeurs par vos identifiants Hostinger.

bash 
$ ssh utilisateur@adresse_ip_du_serveur
$ ssh -p 65002 u734339183@62.72.37.40   # exemple Hostinger avec port personnalisé
Astuce Hostinger : Le port SSH par défaut est 22, mais Hostinger utilise souvent un port personnalisé comme 65002. Vérifiez dans votre panneau hPanel.
Section 01

Navigation & Gestion de Fichiers

Les commandes indispensables pour naviguer dans l'arborescence de votre serveur et manipuler les fichiers.

ls — Lister le contenu d'un répertoire

Affiche les fichiers et dossiers du répertoire courant.

bash 
$ ls                # liste simple
$ ls -l             # liste détaillée
$ ls -la            # avec fichiers cachés
$ ls -lh            # tailles lisibles (Ko, Mo)
$ ls -R             # liste récursive

cd — Changer de répertoire

Permet de se déplacer dans l'arborescence du serveur.

bash 
$ cd /home/u734339183/domains/   # chemin absolu
$ cd public_html                  # chemin relatif
$ cd ..                           # remonter d'un niveau
$ cd ~                            # retour au dossier home

pwd — Afficher le répertoire courant

bash 
$ pwd
/home/u734339183/domains/monsite.com/public_html

mkdir — Créer un répertoire

bash 
$ mkdir mon-dossier
$ mkdir -p parent/enfant/sous-dossier   # crée toute la hiérarchie

rmdir — Supprimer un répertoire vide

bash 
$ rmdir mon-dossier-vide
Section 02

Manipulation de Fichiers

Copier, déplacer, renommer et supprimer des fichiers sur votre serveur Hostinger.

touch — Créer un fichier vide

bash 
$ touch fichier.txt
$ touch index.html style.css script.js   # plusieurs fichiers

cp — Copier des fichiers

bash 
$ cp fichier.txt copie.txt
$ cp -r dossier/ /backup/          # copie récursive d'un dossier

mv — Déplacer ou renommer

bash 
$ mv ancien-nom.txt nouveau-nom.txt     # renommer
$ mv fichier.txt /var/www/html/          # déplacer

rm — Supprimer des fichiers

bash 
$ rm fichier.txt
$ rm -r dossier/         # supprimer un dossier et son contenu
$ rm -rf dossier/        # forcer sans confirmation
Attention : rm -rf est irréversible. Vérifiez toujours le chemin avant d'exécuter cette commande, surtout sur un serveur de production.
Section 03

Lecture & Édition de Fichiers

cat — Afficher le contenu d'un fichier

bash 
$ cat wp-config.php
$ cat -n fichier.txt           # avec numéros de ligne

head / tail — Début ou fin d'un fichier

bash 
$ head -n 20 fichier.txt       # 20 premières lignes
$ tail -n 50 error.log         # 50 dernières lignes
$ tail -f /var/log/syslog      # suivi en temps réel

nano — Éditeur de texte en terminal

bash 
$ nano wp-config.php
# CTRL+O = sauvegarder | CTRL+X = quitter | CTRL+W = rechercher

vi / vim — Éditeur avancé

bash 
$ vi fichier.txt
# i = mode insertion | Esc = mode commande | :wq = sauvegarder et quitter | :q! = quitter sans sauvegarder
Section 04

Recherche & Filtrage

find — Rechercher des fichiers

bash 
$ find . -name "*.php"                # trouver par extension
$ find . -type f -mtime -7            # modifiés ces 7 derniers jours
$ find . -size +10M                   # fichiers de plus de 10 Mo
$ find . -type f -name ".htaccess"    # trouver les .htaccess

grep — Rechercher dans le contenu

bash 
$ grep "motif" fichier.txt
$ grep -r "eval(" --include="*.php" .   # recherche récursive
$ grep -i "error" /var/log/syslog       # insensible à la casse
$ grep -c "404" access.log              # compter les occurrences

locate — Recherche rapide par index

bash 
$ locate wp-config.php
$ sudo updatedb              # mettre à jour l'index de recherche
Section 05

Permissions & Propriété

Essentiel pour la sécurité de votre hébergement Hostinger.

chmod — Modifier les permissions

bash 
$ chmod 755 dossier/              # dossiers web
$ chmod 644 fichier.php           # fichiers PHP
$ chmod 600 wp-config.php         # fichiers sensibles
$ chmod +x script.sh              # rendre exécutable
Rappel : 7=rwx | 6=rw- | 5=r-x | 4=r-- — ordre : propriétaire / groupe / autres

chown — Changer le propriétaire

bash 
$ chown www-data:www-data fichier.php
$ chown -R www-data:www-data /var/www/html/   # récursif
Section 06

Transfert de Fichiers

Transférer des fichiers entre votre machine locale et votre serveur Hostinger.

scp — Copie sécurisée via SSH

bash 
$ scp fichier.txt user@ip:/chemin/distant/       # local → serveur
$ scp user@ip:/chemin/fichier.txt ./local/        # serveur → local
$ scp -r dossier/ user@ip:/chemin/distant/        # dossier complet

wget — Télécharger depuis une URL

bash 
$ wget https://wordpress.org/latest.tar.gz
$ wget -O nom-fichier.zip https://exemple.com/archive.zip

curl — Requêtes HTTP

bash 
$ curl -I https://monsite.com               # headers HTTP
$ curl -o fichier.zip https://url/archive   # télécharger
Section 07

Archives & Compression

Gérer les sauvegardes et archives sur votre serveur.

tar — Créer et extraire des archives

bash 
$ tar -czf backup.tar.gz public_html/     # créer une archive
$ tar -xzf backup.tar.gz                  # extraire
$ tar -tzf backup.tar.gz                  # lister le contenu

zip / unzip — Archives ZIP

bash 
$ zip -r sauvegarde.zip public_html/
$ unzip sauvegarde.zip
$ unzip sauvegarde.zip -d /chemin/destination/
Section 08

Surveillance Système

Surveiller les ressources de votre serveur Hostinger.

df — Espace disque

bash 
$ df -h                       # espace disque en format lisible
$ df -h /home/                # espace d'une partition spécifique

du — Taille des dossiers

bash 
$ du -sh public_html/         # taille totale d'un dossier
$ du -sh * | sort -rh         # trier par taille décroissante

top / htop — Processus actifs

bash 
$ top                         # moniteur interactif (q pour quitter)
$ htop                        # version améliorée (si installé)
$ ps aux                      # liste de tous les processus
$ ps aux | grep php           # filtrer par nom

free — Mémoire RAM

bash 
$ free -h                     # mémoire en format lisible

uptime — Durée de fonctionnement

bash 
$ uptime                      # temps de fonctionnement + charge serveur
Section 09

Réseau & Diagnostic

ping — Tester la connectivité

bash 
$ ping -c 4 google.com       # 4 paquets
$ ping -c 4 monsite.com      # tester son propre site

netstat / ss — Ports ouverts

bash 
$ ss -tulpn                  # ports en écoute
$ netstat -tulpn             # alternative (si installé)

hostname — Informations serveur

bash 
$ hostname                   # nom du serveur
$ hostname -I                # adresse IP du serveur
$ uname -a                   # infos système complètes
Section 10

Gestion des Processus

kill — Arrêter un processus

bash 
$ kill 1234                  # arrêter le processus PID 1234
$ kill -9 1234               # forcer l'arrêt

systemctl — Gestion des services

bash 
$ sudo systemctl status nginx       # statut du service
$ sudo systemctl restart nginx      # redémarrer
$ sudo systemctl stop nginx         # arrêter
$ sudo systemctl start nginx        # démarrer
